# Makani Garage Locator System

## Features
- **User-friendly interface**: Easy navigation and interaction.
- **Locate garages**: Find available garages in real-time.
- **Search functionality**: Search garages based on location and availability.

## Installation
1. **Clone the repository**
   ```bash
   git clone https://github.com/vicmbata98/Makani-Garage-Locator-system-PHP.git
   ```
2. **Navigate to the project directory**
   ```bash
   cd Makani-Garage-Locator-system-PHP
   ```
3. **Install necessary dependencies** (if applicable)
   ```bash
   composer install
   ```

## Usage
- Launch the application by running the following command in the project directory:
  ```bash
  php -S localhost:8000
  ```
- Open your web browser and navigate to `http://localhost:8000`
